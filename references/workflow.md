# Fanqie current workflow notes

Known from user report:

1. Open writer backend
   - https://fanqienovel.com/main/writer/?enter_from=author_zone
2. Enter workspace / editor page
3. Click top-right `下一步`
4. In modal: choose `是否使用AI` -> `否`
5. Confirm publish
6. For scheduled release: click `定时发布` and set date/time
7. Important platform constraint: if the backend warns `请在发布时间前30分钟提交修改内容，否则无法完成修改`, the scheduled chapter is too close to its publish time to safely modify
